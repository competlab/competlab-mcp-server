# Privacy

## How CompetLab Handles Your Data

CompetLab is committed to protecting your data. Here's a summary of what applies to MCP server usage:

**What we collect during MCP interactions:**
- API key (for authentication and workspace scoping)
- Tool calls and parameters (project IDs, competitor IDs, filter parameters)
- Request metadata (timestamps, client type)

**What we don't collect:**
- No conversation content or user prompts from your MCP client
- No personal data beyond what's needed for authentication
- No data from your local machine or development environment

**Data scoping:**
- All data returned by MCP tools is scoped to your organization's workspace
- You can only access projects and competitors that belong to your organization
- API keys are organization-wide — all team members share the same data access

**Competitive data sources:**
- CompetLab monitors publicly available information (websites, sitemaps, pricing pages)
- AI Visibility data comes from querying public LLM APIs (OpenAI, Anthropic, Google)
- No private or behind-login content is accessed
